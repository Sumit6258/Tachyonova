## Overview

**Tachyonova** is a conceptual cryptocurrency and blockchain research project inspired by the theoretical physics of *tachyons* — hypothetical particles that are constrained to always travel faster than the speed of light. In standard relativity, a tachyon possesses imaginary rest mass, meaning it has no lower velocity bound at `c`. It exists, by definition, only in the superluminal regime.

This repository contains the first stage of an ongoing research and design initiative asking a single foundational question:

> *What would a distributed ledger look like if its architecture were governed not by computational speed, but by the geometry of spacetime itself?*

Tachyonova is not a product. It is not a live financial system. It is a rigorous conceptual exploration at the intersection of theoretical physics, cryptographic research, and next-generation distributed systems design.

## The Physics

Tachyonova's conceptual architecture draws from three areas of theoretical physics:

**1. Tachyon Kinematics**  
A tachyon's dispersion relation is `E² = p²c² − |m|²c⁴`. Unlike ordinary matter, its momentum *decreases* as velocity increases. There is no energy cost to move faster — only slower. This inverted energy-velocity relationship forms the conceptual basis for the network's consensus model.

**2. Minkowski Spacetime**  
All events in the Tachyonova model are mapped onto a Minkowski diagram. Normal blockchain transactions occupy timelike worldlines. Tachyonova's theoretical state propagation operates along *spacelike* worldlines — outside the light cone, where causality is not sequential.

**3. Quantum Field Superposition**  
The *Tachyon Core* — the conceptual engine of the protocol — treats state transitions as quantum field events rather than sequential computations. Consensus is not achieved by voting. It emerges from simultaneous field collapse across the Crystal Node lattice, analogous to entangled particles measuring the same observable.

## Conceptual Architecture

### QPoS — Quantum Proof-of-Stake

Tachyonova's theoretical consensus mechanism. QPoS treats each validator not as a sequential processor, but as a fixed point in a quantum field lattice. Key theoretical properties:

- **Zero-latency finality** — state is not voted on; it is collapsed into simultaneously
- **Crystal Node lattice** — 144-node hexagonal validator network modelled on crystalline field geometry  
- **Imaginary-mass incentives** — stake weighting derived from tachyonic energy-momentum relations
- **Retrocausal consistency** — the network's state model permits `Δt → 0` settlement windows

> These properties are **theoretical and conceptual**. They describe a model under research, not a deployed system.
